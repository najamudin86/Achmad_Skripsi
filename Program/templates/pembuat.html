KLASIFIKASI CITRA WILAYAH PERKOTAAN MENGGUNAKAN CONVOLUTIONAL NEURAL NETWORK

Achmad Najamudin Anwar

NIM : 535190047

Mahasiswa Program Studi Informatika Universitas Tarumanagara yang mengembangkan sistem klasifikasi citra wilayah perkotaan menggunakan metode Convolutional Neural Network (CNN) dengan arsitektur EfficientNet-B2.

Penelitian ini bertujuan untuk membantu proses identifikasi karakteristik wilayah perkotaan secara otomatis ke dalam kategori wilayah komersial, pemukiman, dan ruang terbuka hijau.

Website ini dibangun menggunakan framework Flask sebagai media implementasi model CNN sehingga pengguna dapat melakukan klasifikasi citra secara mudah melalui antarmuka web.
